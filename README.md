# Ball-Beam-System
Implementation of a PID controller using TM4C1294NCPDT microcontroller for a Ball&amp;Beam system. Programmed using bare metal code.
This project was programmed using Code Composer.
The following components were used:
-Tiva TM4C1294NCPDT
-VL53L0X ToF sensor
-MG995 Servo
-SPI OLED display
